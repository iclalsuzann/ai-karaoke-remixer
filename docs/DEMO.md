# Demo Script

1. `docker compose up --build` komutuyla backend (FastAPI) ve frontend (Next.js) ayağa kalkar.
2. Tarayıcıda `http://localhost:3000` açılır.
3. “Mikrofonu Aç ve Gönder”e bas: tarayıcı izin ister, sonra durum “recording” olur.
4. Konuş veya bir şarkı mırıldan: RTT metriği ~300-400 ms görünür, “Before/After” player echo sesini çalar.
5. Roadmap: stil transferi ve ışık koreografisi eklenecek (README’de detay).

Sunum tüyosu: Kısa bir 20 sn’lik ekran kaydı al, README’deki diyagramı ekranda gösterip echo prototipini canlı dene.
